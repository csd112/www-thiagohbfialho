Glossary

  • HTML elements - Consist of a opening tag, closing tag, content, and may contain other elements
  • HTML attributes - An HTML element may also have attributes, which contain information about the element and can be used to identify an element
  • head - Contain non-display information such as page title, styles, scripts, etc.
  • body - Contain all displayed content
  • Character references - Is a code that represents a character.
